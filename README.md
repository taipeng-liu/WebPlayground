# Note for Ruby on Rails

* Start the server

```
rails s
```

* Check all routes

```
rails routes
```

* Generate a new page

```
rails g PAGE_NAME
```

* Generate a table

```
rails g scaffold TABLE_NAME COL:DATATYPE COL:DATATYPE
rails db:migrate
```
